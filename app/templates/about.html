About Sentiment Analyzer

Analyze text sentiment with accuracy and ease

Project Overview

This project is a web application built with FastAPI that allows users to submit text and get detailed sentiment analysis results. It uses the VADER Sentiment Analyzer from NLTK to classify text into categories like Very Positive, Positive, Neutral, Negative, and Very Negative.

Additionally, the application can act as an API to programmatically test comments or text for sentiment analysis, making it useful for integration with other applications or automated workflows.

Key Features

  • User registration with session tokens for secure access and rate limiting.
  • Submit text and receive detailed sentiment analysis with a confidence score.
  • Tracks user IP and converts it to location for map visualization.
  • Stores past queries in a database for history and analytics.
  • Interactive map view of sentiment data across different locations.
  • Clean, responsive interface powered by Tailwind CSS.
