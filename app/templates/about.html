About Sentiment Analyzer

Analyze and understand text sentiment with precision and ease

Project Overview

Sentiment Analyzer is a modern web application developed with FastAPI and powered by NLTK's VADER Sentiment Analyzer. It provides real-time emotional classification such as Very Positive, Positive, Neutral, Negative, and Very Negative.

It also offers a clean API layer so developers can integrate automatic sentiment detection into their own applications and workflows.

Key Features

  • User authentication with session tokens.
  • Real-time sentiment analysis with confidence scores.
  • IP ‚Üí Location mapping for sentiment-based geovisualization.
  • Query history stored for analytics and insights.
  • Interactive map view of emotions across cities and countries.
  • Responsive UI powered by Tailwind CSS.

Benefits & Use Cases

  • Businesses analyzing customer feedback.
  • Writers and creators checking emotional tone.
  • Developers building ML-powered applications.
  • Researchers studying sentiment patterns.
